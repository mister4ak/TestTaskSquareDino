# TestTaskSquareDino

Версия Unity - 2020.3.23f1

Build игры сделан под Android

Основные моменты:
1) Инициализация происходит в скрипте Boot.
2) Все префабы загружаются через скрипт AssetProvider, используя папку Resources
3) Скрипт Location отвечает за каждую локацию, на которую приходит игрок, за спавн врагов и выполнение условий, для перехода к следующей локации.
Локация считается зачищенной, если все враги, которые на ней находятся - мертвы (не важно в какой момент их убили) 
4) Для добавления новой локации необходимо добавить в инспектор префаб Location, на нём расположить Waypoint (место, куда становится игрок), 
SpawnMarker'ы (место, где инстанцируются враги, также учитывая rotation) и для каждого указать тип врага.
5) Все добавленные локации необходимо добавить в Boot (в инспекторе).
6) Для создания нового врага необходимо добавить EnemyStaticData (ScriptableObject) по следующему пути - Resources/StaticData/Enemies. 
Префаб врага расположить по пути Resources/Enemy, и указать путь на префаб в скрипте AssetsAddress. Также добавить новый тип врага в EnemyType (enum).
